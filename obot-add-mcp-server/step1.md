# Open the Obot UI

A running Obot instance is provided for you in this environment.

## Open the Admin UI

Open the Obot admin UI in your browser.

From the admin UI you can manage MCP servers, model providers, and agents. In the next step you will add an MCP server from the catalog.

> An **MCP server** exposes a set of **tools** that an AI client can call. Obot hosts these servers and puts a single authenticated **gateway** in front of them, so every tool call is authenticated, authorised, and logged.
