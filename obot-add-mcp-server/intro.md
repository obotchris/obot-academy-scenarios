# Add an MCP Server to Obot

Obot is an open-source platform for implementing MCP (Model Context Protocol) technologies. One of its core features is the **MCP catalog** — a curated set of MCP servers you can host and expose through the Obot gateway.

In this scenario you will:

- Add the **GitHub** MCP server from the built-in catalog
- Configure it with a GitHub Personal Access Token
- Connect an AI client (Claude Desktop) to the Obot gateway
- Make a real tool call and confirm it in the audit log

## Prerequisites

- A GitHub account (to create a Personal Access Token)
- Optional: Claude Desktop, to connect an AI client to the gateway

A running Obot instance is provided for you in this environment — you do not need to install or start anything.
