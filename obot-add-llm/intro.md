# Add an LLM Model Provider to Obot

Obot is an open-source platform for implementing MCP (Model Context Protocol) technologies. It includes a built-in chat client, and to power it you configure one or more **model providers** — the LLMs Obot will use.

In this scenario you will:

- Configure a model provider (OpenAI, Anthropic, or another supported LLM)
- Select a default model
- Test the model in Obot's built-in chat

## Prerequisites

- A valid API key from OpenAI, Anthropic, or another supported model provider

A running Obot instance is provided for you in this environment — you do not need to install or start anything.
