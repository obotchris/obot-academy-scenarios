# Open the Obot UI

A running Obot instance is provided for you in this environment.

## Open the Admin UI

Open the Obot admin UI in your browser.

From the admin UI you can manage model providers, MCP servers, and agents. In the next step you will configure a model provider.

> A **model provider** is Obot's connection to an LLM backend such as OpenAI or Anthropic. You can configure several providers and choose which model powers each agent and the built-in chat.
