# Test the Model in Chat

With a model provider configured, verify it works using Obot's built-in chat.

## Open the Chat

1. In Obot, open the **Chat** view (or click **New Chat**)
2. If prompted, confirm the model you configured is selected

## Send a Prompt

Type a simple prompt to confirm the model responds:

> Hello! In one sentence, what is Obot?

You should get a reply generated by the LLM you just configured. If you configured multiple providers, use the model selector to switch between them and compare responses.

## Troubleshooting

If the chat returns an error:

- Double-check the API key on the **Model Providers** page
- Confirm the provider account has available credit / quota
- Re-open the provider configuration and confirm the key was saved and models loaded
